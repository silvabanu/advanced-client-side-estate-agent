# Estate Agent SPA (Client-side)

## Run
```bash
npm install
npm start
```

## Tests
```bash
npm test
```

## Notes
- Data is in `src/data/properties.json` (7 properties).
- Favourites are stored in localStorage, can be added via star button or drag-and-drop.
- Property page includes gallery (large image + thumbnails + lightbox), and tabs for description, floor plan, and Google map.
